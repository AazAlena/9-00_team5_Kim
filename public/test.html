🏥 Тестирование API

Сервер: http://localhost:3000

📥 1. Импорт данных из CSV

POST /import - загружает doctors.csv, work_slots.csv, appointments.csv

👨‍⚕️ 2. Список врачей

GET /doctors

📅 3. Свободные слоты

GET /slots?doctorId=1&date=2025-05-20

📝 4. Создать запись

POST /appointments

❌ 5. Отменить запись

PUT /appointments/:id/cancel

📊 6. Отчет по утилизации

GET /report/utilization?startDate=...&endDate=...

📋 Результат:

Нажми на любую кнопку
